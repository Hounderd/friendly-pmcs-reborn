**PMC Squadmates adds persistent PMC squadmates to SPT 4.x.** Build a roster, bring selected followers into raids, use Tarkov's voice/gesture command system for basic follower orders, recruit compatible allies, and manage follower gear through Tarkov's social/profile screens. This is an independent SPT 4.x squadmate mod developed around the modern BigBrain, Waypoints, SAIN, and Looting Bots AI stack. **Follower profile integration** ![PMC Squadmates follower profile overview](https://raw.githubusercontent.com/Hounderd/pmc-squadmates/main/assets/readme/friendlypmc-profile-overview.png) **Follower inventory management** ![PMC Squadmates follower inventory management](https://raw.githubusercontent.com/Hounderd/pmc-squadmates/main/assets/readme/friendlypmc-inventory-management.png) **Squad Manager in Messenger** ![PMC Squadmates Squad Manager messenger commands](https://raw.githubusercontent.com/Hounderd/pmc-squadmates/main/assets/readme/friendlypmc-squad-manager-chat.png) ## Requirements Install these before using the mod: - **SPT 4.x** - **BigBrain** - **Waypoints** - **SAIN** - **Looting Bots** The client plugin declares these as BepInEx dependencies because follower movement, targeting protection, combat behavior, and looting support depend on them being loaded together. ## Install Download the latest release ZIP and extract it into your main SPT folder. Your main SPT folder is the folder that contains **EscapeFromTarkov.exe**, **BepInEx**, and **SPT**. Expected final paths: - `BepInEx\plugins\PMCSquadmates.Client\PMCSquadmates.Client.dll` - `BepInEx\plugins\PMCSquadmates.Client\PMCSquadmates.Client.deps.json` - `SPT\user\mods\PMCSquadmates.Server\PMCSquadmates.Server.dll` - `SPT\user\mods\PMCSquadmates.Server\PMCSquadmates.Server.deps.json` - `SPT\user\mods\PMCSquadmates.Server\PMCSquadmates.Server.staticwebassets.endpoints.json` Common install mistakes: - Do not install the server files into a top-level `user\mods` folder. For SPT 4.x, they belong under `SPT\user\mods`. - Do not leave the client plugin inside a loose `netstandard2.1` folder. Keep it under `BepInEx\plugins\PMCSquadmates.Client`. - The included `README.md` and `LICENSE` files can stay in the SPT root or be ignored. They are not loaded by SPT. ## Features Current features: - Persistent server-side follower roster. - Autojoin followers for raids. - Same-side in-raid recruitment into the roster flow. - Messenger Squad Manager commands. - Follower profile integration through Tarkov social UI. - Follower inventory view and gear movement. - Phrase-triggered follower orders for follow, hold, take cover, regroup, loot, and attention/reset. - Configurable heal command hotkey, defaulting to **F10**. - Follower plate overlays for identifying squadmates in raid. - Compatibility handling for common AI-stack conflicts. Current in-raid command input: Follower movement and utility orders are driven by Tarkov phrase triggers, not the old direct follower hotkeys. - **Follow**: FollowMe - **Hold**: HoldPosition, Stop - **Take cover**: GetInCover, CoverMe, NeedCover, TakeCover - **Regroup**: Regroup, NeedHelp - **Loot**: GoLoot, LootGeneric, LootWeapon, LootMoney, LootKey, LootBody, LootContainer, LootNothing, OnLoot, CheckHim - **Attention/reset**: Look - **Heal**: configurable plugin hotkey, default F10 ## Squad Manager **Squad Manager** appears in Messenger and is the main roster interface. Supported commands: - `/help` - Show command help. - `/list` - List roster members. - `/add ` - Create a follower record. - `/rename ` - Rename a follower. - `/delete ` - Remove a follower. - `/autojoin on|off` - Toggle raid autojoin. - `/kit persisted|generated` - Use stored gear or generated gear. - `/equiplist` - List saved player equipment builds. - `/equip ` - Apply a saved build to a follower. - `/gear ` - Show a text summary of stored follower gear. - `/info ` - Show follower state. ![Squad Manager messenger chat commands](https://raw.githubusercontent.com/Hounderd/pmc-squadmates/main/assets/readme/friendlypmc-squad-manager-chat.png) ## Inventory Follower inventory management is handled through the in-game profile flow. 1. Open **Messenger**. 2. Open a managed follower profile from the social/friends flow. 3. Select **OPEN INVENTORY**. 4. Inspect the follower's equipped gear and carry containers. 5. Move items between player and follower inventory. 6. Equip compatible gear into valid follower slots. ![Follower inventory management screen](https://raw.githubusercontent.com/Hounderd/pmc-squadmates/main/assets/readme/friendlypmc-inventory-management.png) ## Known Issues This is an active rebuild, not a finished squad AI overhaul. - Large AI mod stacks can still interfere with follower behavior. - Followers may occasionally stop responding if another behavior layer takes control. - The optional main-menu follower list is disabled for now because it can conflict with SPT's squad interface. - Loot phrase commands require Looting Bots to be loaded. - Inventory drag/drop and profile refresh behavior are still being polished. - BigBrain, Waypoints, SAIN, and Looting Bots are required and must be loaded with the client plugin. When reporting behavior issues, include your SPT version, installed AI mods, whether the follower was fighting/looting/healing/stuck, and any relevant BepInEx or SPT server logs.
